LSS - Layout

Main-Content

This framework uses flexbox layout for simplification, and by default every element inside the container occupies the whole width. In case of separate the elements in columns, is recommended to use the section tag.

Grid-Content

The section creates an one-row grid with 6 columns max. You need to tell how many columns your elements will ocuppy, importing these following classes.

.cols-1 .cols-2 .cols-3 .cols-4 .cols-5 .cols-6

Helper classes

.half-width fills half of the container width

Footer-Section

The footer behaves like main, but all elements are horizontally centralized.
